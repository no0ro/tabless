# Tabless

The goal of this app is to have a temporary place to put open browser tabs. This app allows users to create an account, log in and out, create, update and delete tabs, and view a list of saved tabs.

## Installation & Usage

- Clone repo
- `bundle install`
- `rake db:migrate`
- `shotgun`
